## Dropping real footage into the hero (video-ready scenes)
The hero is a coded scene player (js/cinema.js). Any scene upgrades to real
video automatically: inside its <div class="scene">, add
  <video muted loop playsinline><source src="assets/FILE.mp4" type="video/mp4"></video>
The player detects it, plays it when active, pauses it otherwise.
Ask Assarai for 1920x1080 H.264 MP4, 10–20s, no audio.
